# 前端对接与回归计划

## 对接顺序
1. 鉴权与会话：登录态、权限校验、错误提示。
2. 主题管理：主题 CRUD、状态切换、关键词/竞品聚合显示。
3. 洞察分析：热门笔记、标签、趋势图表。
4. 内容创作：草稿列表、内容详情、标题/图片变体。
5. 运营中心：发布队列、发布状态、评论回复、自动回复开关。
6. 系统设置：账号授权、API Key 配置、自动化参数保存。

## 联调步骤
- 每个模块对接前先跑对应接口测试清单。
- 对接后进行 UI 交互回归（新增/编辑/删除/状态切换）。

## 回归清单
- 鉴权失败提示正确且不泄露敏感信息。
- 主题/内容/发布流程完整闭环。
- XHS 调用失败时 UI 有明确提示并可重试。

## 依赖说明
- 需先完成接口规格文档与鉴权规范。
- 需提供基础测试账号或 mock 模式。
